¿Qué es VisUAM?

Visualiza desde un JSON

Visualiza los formatos disponibles leyendo los datos desde un archivo JSON.

Al tener todos los datos en un archivo json se facilita su lectura, ademas de que este formato está bien estructurado y facil de comprender.

Modelar Resultados

El obtener resultados de sus investigacions y/o practicas en ocasiones requiere de una interpretación visual para hacer más sencilla su comprensión.

No basta con solo ver gráficas de sus datos, la interpretación visual es importante.

¿Requieres gráficos estadísticos?

Si el formato a visualizar lo permite, se pueden observar gráficas de algunos de sus datos.

Nuestro formato de particulas cuenta con esta posibilidad de ver graficos de barras y de pastel.

Empieza a usar el visualizador Ya!

Los miembros de la UAM pueden enviar sus resultados con la estrustura del archivo JSON acordada para poder utilizar el sitio.
